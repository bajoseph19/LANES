Integration Guide

1. Basic Integration

For most recipe pages, simply paste the embed code anywhere in your HTML body. The widget will automatically detect the page URL and fetch ingredients.

2. WordPress Integration

If using WordPress, you can add the code to your theme's single post template or use a Custom HTML widget/block.

<!-- Add to single.php or recipe template -->
<?php if (is_singular('recipe')) : ?>
{{ embed_code | e }}
<?php endif; ?>

3. Testing

After adding the widget code:

  1. Visit one of your recipe pages
  2. Verify the widget appears correctly
  3. Click the widget to test the checkout flow

4. Customization (Optional)

You can pass additional configuration options:

<div id="holistic-market-widget"
     data-api-key="{{ pcp.api_key }}"
     data-theme="light"
     data-position="sidebar">
</div>
